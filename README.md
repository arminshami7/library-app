# My Library App

A simple React application to manage your personal book library.

## Features

- Add books with cover images
- Mark books as read/unread
- Delete books from the library
- Track reading progress (count of read books)
- Image upload for book covers

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```
bash
git clone <repository-url>
```

2. Install dependencies:
```
bash
npm install
```

3. Start the development server:
```
bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```
bash
npm run build
```

## Usage

1. Click the "Choose File" button to select a book cover image
2. Enter the book name in the text field
3. Click "Add Book" to add the book to your library
4. Use the checkbox to mark a book as read
5. Click "Delete" to remove a book from the library
